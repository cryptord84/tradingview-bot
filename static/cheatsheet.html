📈 TradingView
Pine Script fires alert
→
🌐 POST /webhook
Secret validated, rate checked
→
🤖 Claude AI
EXECUTE / REJECT / MODIFY
→
⚡ Jupiter DEX
Quote → Sign → Send
✅ EXECUTE: Trade goes through at suggested size/leverage (capped by risk limits)
🔧 MODIFY: Claude reduces size or leverage, then executes
❌ REJECT: No trade. Logged. Telegram notified.

Duplicate Signal Protection

Same signal_type + symbol + price within duplicate_window_seconds (default 30s) is silently dropped.

Low Balance Shutdown

If SOL balance < low_balance_shutdown_sol, all signals are rejected before reaching Claude.

📍 Position Monitor & Trailing Stop

How It Works

Background service polls every 30s. On BUY execution, creates a position with TP/SL levels derived from ATR. Auto-closes via Jupiter when price hits target.

SettingDefaultDescription
tp_multiplier4.0TP = entry + 4.0 × ATR
sl_multiplier1.5SL = entry − 1.5 × ATR
poll_interval_seconds30Price check frequency
max_close_retries3Retry failed close swaps

Position Statuses

openActive, monitoring
closed_tpTake-profit hit
closed_slStop-loss hit
closed_trailTrailing stop triggered
closed_manualManually closed via dashboard

Trailing Stop-Loss

Locks in profits as price rises. Only ratchets UP — never moves down.

SettingDefaultDescription
activation_atr1.5Trail activates when price > entry + 1.5 × ATR
offset_atr1.0Trail follows at price − 1.0 × ATR

Example

Entry: $130, ATR: $3.50

Fixed SL: $130 − 1.5 × $3.50 = $124.75

Trail activates at: $130 + 1.5 × $3.50 = $135.25

If price hits $140: trail SL = $140 − 1.0 × $3.50 = $136.50

Effective SL = max(fixed, trail) = $136.50

💡 On bot restart, open positions reload from SQLite automatically. No positions are lost.

🧪 Strategy Lab

Dashboard tooling to accelerate the backtest → deploy workflow. All tools live in the Strategy Lab card on the dashboard.

1. Paste Import

Instead of manually entering backtest results, paste the TradingView Strategy Tester summary text and the parser auto-extracts all fields:

Net Profit: $71.00 (+7.10%)
Profit Factor: 1.091
Total Trades: 109
Win Rate: 31.2%
Max Drawdown: $152.00
Sharpe Ratio: 0.342

Fill in Strategy Name, Version, Timeframe, and Symbol, then click PARSE & SAVE to store directly to the backtest database.

2. Generate Alert Indicator

Converts a backtest strategy Pine script into a webhook alert indicator automatically:

  • Strips strategy() and strategy.entry/exit calls
  • Adds indicator(), webhook JSON alert(), cooldown timer
  • Injects deep dip override (configurable ATR depth + relaxed confirms)
  • Adds visual markers (MR BUY / DIP BUY) and info table

Usage: Paste your backtest strategy code, fill in name/version/timeframe, click GENERATE. Copy output to TradingView Pine Editor.

3. Parameter Grid

Generates multiple Pine script variants by sweeping parameter ranges. Define parameter name + min/max/step, paste your strategy code, and get all combinations (max 100).

Example: min_confirm: 3-5 step 1 + bb_mult: 1.5-2.5 step 0.5 = 9 variants. Each variant has the parameter baked into the strategy title for easy identification in TradingView.

4. Token Sweep

Duplicates a strategy across multiple tokens. Select checkboxes (SOL, JTO, WIF, BONK, PYTH, RAY) and click GENERATE. Each variant has the token name appended to the strategy title.

5. Strategy Comparison

Cross-strategy, cross-token analysis from all stored backtests. Shows:

  • Deployable Strategies — PF ≥ 1.0 with 20+ trades (ready for alert indicator conversion)
  • Strategy Averages — aggregate PF, win rate, trade count per strategy
  • By Token — which tokens perform best with which strategy

6. Strategy Templates

Pre-built starting points for new strategies. Click USE to load into the generator or grid/sweep tools.

Arbitrage Strategies

  • YES/NO Spread: Buy both when YES+NO < 100¢ minus fees
  • Bracket Sum: Multi-outcome events where ask prices sum < 100%
  • Crossed Book: Bid > ask anomalies in high-volume markets

Market Maker Strategies

  • Midpoint: Symmetric quotes around orderbook mid
  • VWAP: Anchored to volume-weighted average price
  • Volatility: Dynamic spread widening in choppy markets
  • Inventory skew + adverse selection detection
  • Portfolio-level risk caps across all markets

AI Agent Consensus

  • Analyst: Fundamental probability estimation
  • Contrarian: Crowd sentiment reversal
  • Momentum: Trend following
  • Weighted vote by confidence, min 70% strength to trade
⚠️ All Kalshi bots default to enabled: false and auto_trade: false. Enable manually in config.yaml or via dashboard toggles. Auto-trade requires explicit opt-in per bot.
