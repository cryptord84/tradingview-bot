🚀 Startup Order

1

Activate virtual environment

cd ~/Documents/Claude/Projects/tradingview-bot source venv/bin/activate
2

Verify config exists

ls config.yaml .env # Both must exist
3

Start the bot

python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload

Watch for: Bot initialized successfully

4

Expose webhook publicly (new terminal tab)

ngrok http 8000 # OR cloudflared tunnel --url http://localhost:8000

Copy the https://xxxx.ngrok.io URL → paste into TradingView alert webhook field

5

Open dashboard

open http://localhost:8000

Shutdown

Ctrl+C # in the uvicorn terminal

Run without auto-reload (production)

python -m uvicorn main:app --host 0.0.0.0 --port 8000 --workers 1

🔄 Signal Flow (end-to-end)

📈 TradingView
Pine Script fires alert
→
🌐 POST /webhook
Secret validated, rate checked
→
🤖 Claude AI
EXECUTE / REJECT / MODIFY
→
⚡ Jupiter DEX
Quote → Sign → Send
✅ EXECUTE: Trade goes through at suggested size/leverage (capped by risk limits)
🔧 MODIFY: Claude reduces size or leverage, then executes
❌ REJECT: No trade. Logged. Telegram notified.

Duplicate Signal Protection

Same signal_type + symbol + price within duplicate_window_seconds (default 30s) is silently dropped.

Low Balance Shutdown

If SOL balance < low_balance_shutdown_sol, all signals are rejected before reaching Claude.

ModeHowCost

Claude receives

  • Full TradingView signal
  • Live wallet balance (SOL + USD)
  • SOL market data (price, 24h change, volume)
  • Latest news headlines (NewsAPI / Tavily)
  • All risk parameters
